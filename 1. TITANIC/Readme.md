# 🚢 Titanic Survival Prediction

## 📖 Overview
This project predicts survival of Titanic passengers using machine learning and feature engineering techniques.

## 📊 Dataset
- Source: Kaggle Titanic Dataset
- Features: Age, Sex, Pclass, Fare, Embarked, FamilySize, Title, etc.

## ⚙️ Workflow
- Data cleaning (missing values handling)
- Feature engineering (Title, FamilySize, IsAlone)
- Encoding categorical variables
- Model training and evaluation

## 🤖 Models Used
- Logistic Regression
- Decision Tree
- Random Forest (Best Model)

## 🏆 Kaggle Score
- Public Score: 0.77033  
- Private Score: 0.76510  

## 📈 Results
- Best Accuracy: ~77%
- Improved performance after feature engineering
