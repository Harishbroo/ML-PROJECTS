# Machine Learning Projects Portfolio

Welcome to my Machine Learning repository! This portfolio contains a collection of data science and machine learning projects focusing on exploratory data analysis (EDA), data preprocessing, feature engineering, model implementation, and deployment.

## 📁 Repository Structure

Each folder in this repository represents a standalone machine learning project. Inside each folder, you will find:

- **Jupyter Notebooks (`.ipynb`)** – Step-by-step implementation including data cleaning, feature engineering, visualization, model training, and evaluation.
- **Datasets** – Training/testing datasets (where file size permits).
- **Trained Models (`.pkl`)** – Serialized machine learning models used for deployment.
- **Streamlit Applications (`app.py`)** – Interactive web applications for model predictions.
- **Docker Support** – Dockerfile and requirements for containerized deployment.
- **Project READMEs** – Documentation describing the project's objective, workflow, models used, results, and deployment.

## 🛠️ Tech Stack & Core Tools

- **Programming Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Web Framework:** Streamlit
- **Deployment:** Docker, Render
- **Version Control:** Git, GitHub
- **Development Environment:** Jupyter Notebook, VS Code, Git Bash
