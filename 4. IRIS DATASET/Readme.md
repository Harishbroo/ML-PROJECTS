# 🌸 Iris Flower Classification

## 📖 Overview
This project predicts Iris flower species using a machine learning model and a simple Streamlit web app.

---

## 📊 Dataset
- iris.csv

---

## 📁 Files
- iris.ipynb → Model training and analysis
- iris.csv → Dataset
- iris_model.pkl → Trained model (pickle)
- app.py → Streamlit app

---

## 🤖 Models Used
- Logistic Regression
- KNN
- Decision Tree

---

## 📈 Result
- Accuracy: ~95%+

---

## 🚀 How to Run

```bash
pip install streamlit numpy scikit-learn
streamlit run app.py
